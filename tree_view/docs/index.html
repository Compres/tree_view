Tree View for a DocType

DocType and a Browser Page

Install

From your site

To install this app, login to your site and click on "Installer". Search for Tree View and click on "Install"

Using Bench

Go to your bench folder and setup the new app

$ bench get-app tree_view https://github.com/revant/tree_view
$ bench new-site testsite
$ bench --site testsite install-app tree_view

Login to your site to configure the app.
